## [1.0.0] - 2024-01-09

### Added
- 用户注册功能（公开接口）
- 用户登录功能（JWT 认证）
- 获取当前用户信息接口
- 用户管理功能（管理员）
  - 创建用户
  - 获取用户列表
  - 获取用户详情
  - 获取用户统计数据
  - 删除用户（仅管理员，且只能删除普通用户）
- 权限控制（基于角色的访问控制）
- 前端页面
  - 登录页面
  - 注册页面
  - 仪表盘页面
  - 用户管理页面（管理员）
- 后端 API 文档（Swagger/OpenAPI）
- 单元测试（后端和前端）
- CI/CD 工作流（GitHub Actions）
- 静态代码分析工具集成
- 安全扫描工具集成

### Security
- JWT Token 认证
- bcrypt 密码加密
- 输入验证
- XSS 基本防护

### Changed
- 移除了图片上传模块，简化项目结构
- 优化了 API 响应格式
- 改进了错误处理机制

### Fixed
- 修复了数据库迁移问题
- 修复了用户统计数据显示问题

### Documentation
- 添加了 API 文档
- 添加了项目结构文档
- 添加了用户流程图文档
- 添加了测试计划文档
- 添加了部署文档
- 添加了故障排除文档
